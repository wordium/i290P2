Did you see a difference?

In Cory Doctorow's Metacrap, we learn that people are lazy, and schemas aren't neutral. We hope to demonstrate the difference between completely freeform guesses (the first set of movies) and guesses that have some autocompletion suggestions (the second set of movies). Each set of movies are hand-picked to have similar genres, and while you might have written "action" for one, perhaps for the second, you used the suggested genre from Rotten Tomatoes of "Action & Adventure."
